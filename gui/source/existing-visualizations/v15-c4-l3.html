Caption (for the doc)

The two C4-L3 component diagrams that anchor the architecture chapter. Storage (V15a) shows the read/write paths: AuditWriter is single-writer; tokens go through envelope encryption; Postgres is single-tenant. Security (V15b) shows the request pipeline every entry point shares: authn → authz → confirmation → redaction → rate limit → prompt-injection guard, with every decision audit-hooked. There is no bypass path — that's the point.
