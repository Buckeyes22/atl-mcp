# Security

Report vulnerabilities to **ctanferno22@gmail.com**.

Acknowledgment within 5 business days. Triage within 10. Coordinated disclosure preferred.

For full policy — what's in scope, what to include, severity guidance, response timeline, and what we will / won't do — see [`docs/sdlc/06-security/vulnerability-disclosure.md`](docs/sdlc/06-security/vulnerability-disclosure.md).

For the threat model and controls matrix, see [`docs/sdlc/06-security/threat-model.md`](docs/sdlc/06-security/threat-model.md) and [`docs/sdlc/06-security/controls-matrix.md`](docs/sdlc/06-security/controls-matrix.md).

**Do not file public issues for security vulnerabilities.** Public issues = public disclosure before patch.
