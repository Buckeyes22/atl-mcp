The VcsProvider abstraction. v1 ships exactly one realization (Bitbucket Cloud); GitHub and GitLab classes are sketched as runway only — the abstraction is shaped to absorb them post-v1 without rework. Behavioral parity is enforced by the contract test suite (v6 §31), not just by inheritance — every realization runs the same tests. The same shape applies to JiraProvider and ConfluenceProvider (Atlassian side).
