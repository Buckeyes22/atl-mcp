Caption (for the doc)

The lethal trifecta describes the dominant LLM-app risk class: an operation that reads PRIVATE data, processes UNTRUSTED content, and emits EXTERNALLY in one execution. Each property alone is ordinary; the triple intersection enables exfiltration. atl-mcp's policy decision layer detects the combination and either blocks the operation or routes it to require_approval (per v6 §38.1).

Design notes

Color follows the project's classification + severity semantics: red for PRIVATE, amber for UNTRUSTED, blue for the external boundary.
