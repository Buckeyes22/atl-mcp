## A note for skeptical readers

This portfolio leans hard on the dogfooding frame: a tool that produces project structure has produced its own. If you suspect that frame is sleight-of-hand, the test is to compare:

- The structure of [PCO](https://lateapexllc.atlassian.net/jira/software/projects/PCO/boards/1) (8 epics, 10 components, label taxonomy) against v6 §12 ("Generated Jira Structure").
- The structure of [ACO](https://lateapexllc.atlassian.net/wiki/spaces/ACO) (35 pages, 8 sections) against v6 §11 ("Generated Confluence Structure").

If those match, the dogfooding claim holds. If they don't, the claim is broken and you should grade me down.

For the AI-assisted-vs-authored question, see [`ai-honesty.md`](ai-honesty.md). I address it directly there.
