# Phase 3: Scoping & Proposal

**Purpose:** Convert ambiguity into a priced decision. This is where the engine's pricing intelligence, value framing, and scope control come together.

## Required Inputs
- Completed discovery brief from Phase 2 (or clear problem from Phase 1 for straightforward engagements)

## Required Outputs
- Scoped engagement with explicit deliverables and exclusions
- Three-option proposal (Good/Better/Best per Stark/Enns framework)
- Pricing using appropriate model (fixed/tiered/project/value/retainer)
- Timeline with client dependencies
- Change-order process defined

## Guardrails
- **Via Negativa gate** — challenge every scope item: "Can we remove this and still deliver core value?"
- **Planning Fallacy check** — reference class estimation, 30% buffer requirement
- **METR productivity illusion awareness** — do NOT assume AI speedup for complex tasks (research: developers 19% SLOWER with AI on real tasks, believed 20% faster)
- **Pricing model selection** — matched to engagement type (see engine/pricing/)
- **SMB sweet spot** — initial engagements target $5K-$15K for fastest close (single decision-maker approval)
- **Value language** — frame as business outcome, not deliverable list. "Understanding whether your business could survive a data breach" > "security posture review"

## Paid Discovery Option (Brennan Dunn / Ben Lee Model)

Before a full proposal, consider offering a **paid discovery engagement** as a standalone product. This replaces free proposals with a compensated diagnostic that delivers immediate value AND pre-sells implementation.

- **AI Operations Assessment**: $3,000-$5,000 for a structured 2-4 week diagnostic
- Produces a business-focused report, not a technical document
- Report structure follows Dunn's "assumption-less proposal" framework:
  1. "Why You're Here" — root problem in client's language
  2. "Where You Want To Be" — transformed future state with emotional drivers
  3. "What We Want To Do" — proposed paths with options
  4. "What We'll Be Preparing For" — risks and mitigation
  5. "Why We're Best For The Project" — domain expertise with case studies
  6. "How We Can Do This For You" — tiered pricing with estimated ROI
- Always present **at least two implementation options** — never one ("an ultimatum")
- Conversion to implementation: **50-75%** (Dunn model)
- Ben Lee (Rootstrap): scaled paid roadmapping to **$1M/year** at $15K-$30K per engagement, ~4/month
- The report is valuable even if the client doesn't hire you for implementation

**When to use paid discovery vs free proposal:**
- Complex engagement (>$10K) with unclear scope → paid discovery
- Straightforward fixed-price service with published scope → free proposal (or no proposal needed)
- Client can't articulate what they need → paid discovery is the right first step

## Phase Boundary Checkpoint
**Scoping → Contract:** Is the scope boundary right? Are deliverables verifiable? Is anything missing? Is the pricing defensible? If yes, proceed to Phase 4.

## Pricing Reference (from research synthesis)
- Website reviews: $249-$349 minimum (not $129)
- Security posture: $499-$995 (not $179)
- AI opportunity audit: $5K-$7.5K
- vCISO retainer: $3K-$5K/month, 90-day minimum
- See `engine/pricing/` for full model templates
- Only 17.3% of consultants use value-based pricing; 73% of clients prefer it — immediate differentiator
